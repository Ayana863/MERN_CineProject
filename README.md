CineScope – Movie Discovery & Recommendation Platform
....................................................


Live Demo :  https://movie-discovery-recommendation-plat-roan.vercel.app/

Description
___________

CineScope is a modern, responsive movie discovery web application built for movie lovers. Users can explore trending and upcoming movies, browse by genre, manage favorites and watch history, and share their own reviews with star ratings.



Features
________

*User authentication (Firebase)
*Movie & series cards with posters and ratings.
*Trending, Popular & Top-Rated sections (TMDB API).
*Genre-based filtering (Action, Drama, Comedy, Thriller, etc.).
*Advanced search with real-time results.
*Detailed movie modal (overview, rating, poster).
*Favorites & Watch History (saved using Local Storage).
*User Review System (rating + written review).
*Fully responsive design for all devices.
